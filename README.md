# Claude Code Research Plugins

Specialized collection of development plugins designed for expert analysis and research workflows.

## 📦 包含的插件

### Evolutionary Biology Expert Analysis v0.1.0

专业的进化生物学研究者分析系统，通过先进的文献挖掘和网络分析技术进行专家思维图谱重构、学术网络分析和研究影响评估。

#### 功能特性
- 🧠 思维图谱重构
- 🌐 网络中心性分析  
- 👥 合作网络映射
- 📊 引用网络分析
- 🎯 研究影响评估
- 🔍 多源数据验证

#### 技能模块
- 专家网络映射
- 学术文献分析
- 搜索词优化
- Google Scholar分析
- 时间动态分析
- 批判性思维分析
- 上下文感知分析

## 🚀 安装

### 依赖的MCP服务器
```bash
claude mcp add article-mcp uvx article-mcp server
claude mcp add mediawiki-mcp-server npx @professional-wiki/mediawiki-mcp-server@latest
claude mcp add sequentialthinking npx -y @modelcontextprotocol/server-sequential-thinking@latest
claude mcp add playwright npx @playwright/mcp@latest --browser chrome --headless
```

### 可选环境变量
```bash
export EASYSCHOLAR_SECRET_KEY="your_secret_key"
```
