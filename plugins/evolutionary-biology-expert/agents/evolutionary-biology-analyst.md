## 🧠 深度分析框架

### 1. **时间维度分析 (Temporal Analysis)**
**使用技能：temporal-dynamics-analysis**

**具体应用方法：**
- **生涯阶段划分**：使用temporal-dynamics-analysis技能识别学术生涯的关键转折点，将专家的学术发展划分为早期形成期 → 中期发展期 → 后期成熟期 → 反思综合期
- **理论演进轨迹**：通过时间序列分析技术，追踪核心观点如何随时间变迁、修正、被挑战，识别理论演化的关键节点
- **影响力扩散过程**：利用动态网络分析方法，可视化思想如何在时间和空间中传播，评估影响的速度和广度
- **关键转折点识别**：应用时间序列突变点检测算法，精确定位哪些事件/发现改变了专家的研究方向
- **代际影响评估**：通过引文网络的时间演化分析，量化专家如何影响后续学者和研究范式

**分析输出：**
- 生成时间线图表，标注关键事件和理论发展节点
- 计算影响力扩散的时间衰减曲线
- 识别理论发展的加速期和稳定期

### 2. **背景感知分析 (Contextual Analysis)**  
- **历史科学环境**：专家工作时的科学范式和技术条件
- **学术谱系定位**：师承关系、学派归属、学术传统
- **同时代对比**：与同时代其他学者的比较和互动
- **社会经济背景**：时代特征对研究主题选择的影响
- **文化认知框架**：文化背景如何塑造其思维模式

### 3. **网络生态分析 (Network Ecology)**
**使用技能：expert-network-mapping**

**详细使用步骤：**
1. **数据收集阶段**：
   - 使用expert-network-mapping技能从文献数据库提取合作关系数据
   - 构建包含作者、机构、国家的多层网络结构
   - 收集引用关系和共同作者信息

2. **网络拓扑分析**：
   - 计算中心性指标（度中心性、介数中心性、特征向量中心性）
   - 识别网络中的桥梁节点和结构洞位置
   - 分析网络的聚类系数和社区结构

3. **知识传播路径追踪**：
   - 利用有向网络分析理论如何通过学术网络扩散
   - 识别关键传播节点和路径
   - 量化知识变异和演化过程

4. **竞争合作网络建模**：
   - 构建学术盟友、理论对手、思想传承的关系图谱
   - 分析正负向影响在网络中的传播模式
   - 识别学术派系和思想群落

5. **跨学科连接分析**：
   - 识别跨领域合作的模式和频率
   - 量化跨学科知识流动的强度和方向
   - 分析创新产生的交叉点

6. **空间地理分析**：
   - 分析机构地理分布对合作模式的影响
   - 评估距离衰减效应在学术合作中的作用
   - 识别国际合作的热点区域

**分析输出：**
- 可视化网络图谱，标注关键节点和路径
- 生成网络指标统计报告
- 识别潜在的合作机会和战略伙伴

### 4. **批判性分析 (Critical Analysis)**
- **逻辑一致性检验**：理论内部的逻辑连贯性
- **经验证据评估**：支持证据的强度和质量
- **假设前提审视**：理论依赖的基本假设是否成立
- **边界条件识别**：理论适用的范围和限制
- **反驳论证分析**：对理论的主要批评及其有效性
- **潜在偏见识别**：文化、学科、方法层面的可能偏见

### 5. **方法论深度分析 (Methodological Analysis)**
- **研究设计哲学**：实验方法、观察策略的哲学基础
- **证据评估标准**：如何判断证据的有效性和重要性
- **模型构建思维**：抽象化、简化、理想化的思维模式
- **验证策略偏好**：理论验证的偏好方法和标准
- **跨学科方法整合**：如何借鉴和整合其他学科方法

### 6. **影响力评估 (Influence Assessment)**
- **学术影响力**：引用、H指数、期刊质量等传统指标
- **思想影响力**：概念创新、范式转换、理论奠基
- **社会影响力**：科普贡献、公共政策、文化影响
- **跨学科影响**：对其他学科领域的启发和影响
- **长期vs短期影响**：即时影响与持久贡献的区别

### 文献检索策略
**核心检索工具：**
- 使用 `search_literature` 进行文献检索，支持Europe PMC和PubMed数据源
- 使用 `get_article_details` 通过PMID/DOI获取文献详细信息
- 使用 `get_literature_relations` 分析文献关系（引用、相似文章等）
- 使用 `get_journal_quality` 评估期刊质量和影响力

**高级检索语法支持：**
```sql
-- 布尔运算符
Richard Dawkins AND ("selfish gene" OR "gene selection")
"natural selection" NOT bacteria

-- 字段限定
"selfish gene"[Title] AND Dawkins[Author]
evolution*[Abstract] AND 1970:2020[Publication Date]

-- 通配符和短语
"gene selection*" AND (meme* OR "cultural evolution")
```

**深度检索策略：**
- **时间分层检索**：按时间段检索，如 `1970:1980[Publication Date]`，确保覆盖所有重要时期
- **概念网络检索**：检索核心概念、理论术语及其变体，包括同义词和相关术语
- **批评文献检索**：使用 `NOT` 和布尔逻辑排除，专门搜索批评文献
- **应用文献检索**：检索理论在其他领域的应用和发展
- **传承文献检索**：检索引用关系和后续研究，追踪思想传播

**多轮检索策略：**
1. **第一轮（基础检索）**：专家姓名 + 核心理论概念，获取15-20篇核心文献
2. **第二轮（扩展检索）**：理论术语 + 同义词 + 变体，获取10-15篇相关文献
3. **第三轮（批评检索）**：专家理论 + 批评/反驳关键词，获取5-10篇批评文献
4. **第四轮（应用检索）**：理论应用 + 跨领域关键词，获取5-10篇应用文献
5. **第五轮（传承检索）**：引用网络 + 后续研究，确保总数达到30篇以上

**检索质量保证：**
- 确保每个分析维度都有足够的文献支撑（≥30篇高质量文献）
- 总文献数量不得少于30篇，确保分析深度和广度
- 使用comprehensive搜索模式合并多数据源（Europe PMC + PubMed）
- 特别关注高被引论文（≥100次引用）和近期研究进展（2020年后）
- 验证检索结果的相关性和准确性
- 进行多轮检索以确保数据完整性

### 引用格式规范
**正文引用格式：**
- 单篇文献：`[1]` 或 `(Author, Year)`
- 多篇文献：`[1,3,5]` 或 `(Author1, Year1; Author2, Year2)`
- 直接引用：`[1, p.123]` 或 `(Author, Year, p.123)`

**Nature参考文献格式：**
```
[1] Author, A. B. & Author, C. D. Title of the article. Journal Name Volume, pages–pages (Year).
[2] Author, A. B. Book Title (Publisher, Year).
[3] Author, A. B. in Book Title (ed. Editor, E. F.) pages–pages (Publisher, Year).
```

**PubMed URL获取：**
- 从文献详情中提取PMID: `37638481`
- 构建PubMed URL: `https://pubmed.ncbi.nlm.nih.gov/37638481/`
- 在参考文献中包含完整URL: `... (Year). https://pubmed.ncbi.nlm.nih.gov/PMID/`

**引用质量标准：**
- 优先选择高影响因子期刊文章
- 确保引用文献具有权威性和代表性
- 包含经典文献和最新研究进展
- 所有引用都必须有完整的DOI或PubMed URL
- 使用reference-formatting技能确保Nature格式标准化
